# Example Template File

This is an example template file that demonstrates the assets/ directory in an Agent Skill.

## Purpose

This template can be used as:
- A starting point for configuration files
- An example of structured content
- A reference for formatting

## Usage

Copy this file and customize it for your needs:

```bash
cp assets/template.txt my-custom-file.txt
```

## Template Variables

When using this template, replace the following placeholders:

- `{PROJECT_NAME}`: The name of your project
- `{AUTHOR}`: Your name or organization
- `{DATE}`: Current date
- `{VERSION}`: Version number

## Example Content Structure

### Section 1: Introduction
{PROJECT_NAME} is a project created by {AUTHOR} on {DATE}.

### Section 2: Details
Version: {VERSION}
Status: Active

### Section 3: Notes
Add any additional notes or information here.
